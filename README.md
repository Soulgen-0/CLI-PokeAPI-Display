# CLI-PokeAPI-Display
## Description
  This is a Command-Line-Interface project/app developed for practice. 
  It allows the user to query for data on a Pokemon via PokeAPI.
  Information is then displayed to the user.
## Languages Used
  This project is built in Python
## Features
  - Query data on a specified Pokemon species
  - API Fetch Requests
  - JSON parsing on received requests
  - Error handling of bad/corrupt requests
## Goals
  I hope to learn about external API usage and requests.
  This is new territory for me and I hope to understand it better.
