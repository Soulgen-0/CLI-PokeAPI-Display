# CLI-PokeAPI-Display
## Description
  This is a Command-Line-Interface project/app developed for practice. 
  It allows the user to query for data on a Pokemon via PokeAPI.
  Information is then displayed to the user.
## Languages Used
  This project is built in Python
## Features
  - Query data on a specified Pokemon species
  - API Fetch Requests
  - JSON parsing on received requests
  - Error handling of bad/corrupt requests
### Querying
  When at the query screen, any species of pokemon 
  can be typed in to fetch data on it.
### Versions
  When at the version screen, any version can be specified 
  to view the pokemon description for that game version.
### Descriptions
  Appropriate game version descriptions are displayed for each pokemon
  depending on the chosen version.
### Error Screen
  If any error occurs, an error screen is displayed to show what went wrong.
### Returning
  When on any screen, pressing enter with no input returns to the previous screen.
